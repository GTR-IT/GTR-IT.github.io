## Token Usage — Lessons Learned
- Today's session was very long and token-heavy
- Key culprits: long conversation history, browser ARIA snapshots (~1-3k tokens each), failed attempts staying in context, security notice boilerplate on every snapshot
- Rule going forward: **spawn sub-agents for heavy browser automation tasks** (e.g. document creation, long research)
- Prefer JS `evaluate` over full ARIA snapshots where possible
- Prefer web_fetch over browser where possible
- Start fresh sessions more often
